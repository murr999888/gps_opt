Основные понятия. Допустимые группы клиентов.

При первом запуске или при обновлении данных таблицы "Машины", для всех машин устанавливаются в качестве доступных все группы клиентов.

Для каждой машины устанавливается список допустимых групп клиентов. 

К примеру, водовоз работает по оптовым клиентам и предприятиям.

Поскольку в свойствах машины установлено значение максимальной загрузки для баллонов, которые могут заказать любые клиенты, то программа может распределить на эту машину в том числе заказы частных клиентов.

Чтобы этого избежать группы "Бесплатники", "Реклама" и "ЧАСТНЫЕ ЛИЦА (адреса)" - отключены (снят флажок на данной строке).

Внимание!

Допустимые машины клиента имеют приоритет перед допустимыми группами клиентов машины!

Например, если для Исузу №1 установлены допустимые группы "Оптовики", "Предприятия", но при этом в заказе для клиента группы "Частные лица-Адреса" эта машина указана в качестве допустимой, то заказ может быть распределен на эту машину.

Особенности работы с веб-интерфейсом:

  1. Настройки веб-страницы сохраняются только для браузера, в котором эта страница открыта.
  2. Данные веб-страницы не сохраняются в период между открытиями (или обновлениями) страницы. За исключением таблицы данных автомобилей и пробок.
  3. Данные таблиц, которые получены из учетной базы данных не обновляются автоматически. Если после загрузки списка заказов из учетной базы данных, там произошли изменения (добавлен / изменен / удален заказ), то эти изменения не будут отображены в веб-интерфейсе немеделенно, а только после повторной загрузки данных.
  4. При загрузке данных из учетной базы данных, все внесенные на веб-странице изменения будут утеряны.

Настройка:

Если при наведении курсора на вертикальную границу таблицы появляется индикатор , то вы можете зажав левую кнопку мыши, сдвинуть границу вправо-влево и таким образом получить больше или меньше места для отображения содержимого таблицы (например при нехватке места для столбцов таблицы, когда их содержимое обрезается). Таким же способом изменяется ширина столбца.

Некоторые столбцы имеют меню. Если при наведении курсора на заголовок столбца на нем начинает отображаться черный треугольник меню, то для этого столбца доступны функции сортировки и / или скрытия.

Чтобы скрыть столбец, снимите соответствующий флажок.

Отсортировать таблицу по нужному столбцу можно через это же меню или простым нажатием на заголовок столбца (если сортировка доступна, на рисунке ниже для столбца "Водитель" эта функция отключена).

 

Для некоторых панелей предусмотрен режим сворачивания / разворачивания, который позволяет более экономно использовать место на экране.

Заголовок такой панели имеет соответствующую кнопку (стрелкой показано направление, в котором будет свернута панель). Как правило, если панель имеет способность сворачиваться, то свернуть ее можно просто кликнув мышью по заголовку панели.

 

В свернутом виде панель представлена на экране своим заголовком. Развернуть ее можно нажав мышью на значок разворачивания. Или кликнуть на заголовок панели.

 

Панель может сворачиваться / разворачиваться как в горизонтальном, так и вертикальном направлении.

Составные панели (содержащие несколько таблиц)  как правило сделаны в виде "аккордеона": обычно открыта одна панель, остальные свернуты.

Для сворачивания / разворачивания таких панелей нужно кликнуть мышью на соответствующую иконку или на заголовок панели.

Табличные панели обычно имеют набор элементов управления для манипулирования данными таблицы. Функции кнопок описаны в соответствующих разделах этой справки.

Веб-интерфейс запоминает размеры и расположение панелей, состав и размер колонок таблиц.

Для того, чтобы установить / снять флажки во всей таблице, кликните на флажок в заголовке колонки таблицы

Основные понятия. Пробки.

Существуют ситуации, когда на отдельных участках дорог необходимо ограничить скорость или полностью заблокировать движение транспортных средств:

  • участок дороги закрыт на ремонт или реконструкцию;
  • пробка пропускным пунктом на территорию завода или данный въезд закрыт для наших автомобилей;
  • расчетная скорость движения по данным карты не соответствует реальному состоянию дороги и т.п.

Для учета подобных ситуаций при расчете маршрутов служит функционал вкладки "Пробки".

Таблица "Пробки" в отличие от остальных таблиц веб-интерфейса является общей для всех пользователей.

Мы отмечаем на карте участок дороги и устанавливаем для него скорость движения.

При достаточно малых величинах скорости (1-5 км/ч) и достаточной протяженности участка дороги, при проведении расчета программа будет прокладывать маршруты избегая такие места. 

Важно помнить об нюансах, связанных с техническими ограничениями маршрутизатора OSM-карт:

  1. Блокирование (скорость 0 км/ч) подъездной дороги к месту расположения клиента, сделает его полностью недоступным для участия в расчете.
  2. Возможно построение маршрута, который пересекает дорогу, для которой установлено какое-либо (даже 0 км/ч) ограничение скорости.
  3. Обновление данных маршрутизатора происходит только по запросу (кнопка "в OSRM") и может занимать несколько минут.

Пример пересечения маршрута и пробки:

  1. Несмотря на ограничение скорости в 1 км/ч маршрут будет проложен частично через пробку с пересечением другой пробки, поскольку это единственный заезд к клиенту. 
  2. Пересечение с пробкой возможно и не считается маршрутизатором непреодолимым препятствием.

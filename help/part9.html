Основные понятия. Заправки.

При расчете маршрутов возможно учесть необходимость заправки машин, т.е. добавить в маршруты служебные заказы, связанные с посещением АЗС.

В программе заложено два режима расчета заправок:

"Сначала заправить" - для нужной машины вручную указывается АЗС, которую машине необходимо посетить перед началом рейса.

"По расходу топлива" - маршрут для машины при необходимости будет проложен через подходящую АЗС с учетом индивидуальных параметров машины и расположения заказов маршрута. 

Для этого в параметрах машины задаются:

  • емкость бака и расход топлива;
  • объем топлива перед началом рейсов и необходимый минимальный остаток в баке в конце дня.

Индивидуальные параметры задаются на вкладке "Топливо":

Важно сразу установить тип топлива для машины. Если машина использует газовое топливо, установите флажок "Топливо - газ".

Список доступных АЗС

Прежде чем использовать учет заправок, необходимо заполнить список АЗС:

На панели инструментов таблицы машин нажмите кнопку "Заправки"

Откроется окно диалога работы со списком заправок:

При первом запуске список заправок пуст и его нужно загрузить из учетной системы. Нажмите кнопку "Получить"  на панели инструментов:

После загрузки все АЗС выделены флагом "Участвует в расчете". Чем больше доступных заправок участвует в расчете, тем сложнее и дольше будет происходить расчет. Поэтому обозначьте флагами минимально необходимый список заправок. Если есть машины работающие на газе - отметьте нужное количество газовых заправок.

Установите при необходимости интервал времени, необходимого для проведения заправки. По умолчанию это 10 минут. Этот параметр аналогичен параметру "Время разгрузки" для обычного заказа.

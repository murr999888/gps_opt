Основные понятия. Главное депо.

Главное депо - это место базирования автомобилей, начало и конец любого маршрута (рейса), место из которого выезжают машины утром и куда возвращаются вечером.

В главном депо производится погрузка машин перед выездом в рейс товаром в количестве, равном сумме товара, отгружаемого по всем заказам маршрута, и производится разгрузка машин о товаров, в количестве равном сумме всех товаров, загружаемых по заказам маршрута.

Погрузка в начале маршрута учитывает товары, которые предполагается загрузить на заказах маршрута. Разгрузка в конце маршрута учитывает товары, которые предполагается разгрузить на заказах маршрута.

Главное депо и пункты промежуточной загрузки отображаются на вкладке "Формирование МЛ"

Форма настройки параметров главного депо:

Емкость депо.

Емкость депо определяет ограничение количества товаров определенного вида, которые возможно "загрузить" на машины в начале рейса при расчете маршрутов. Если суммарное количество товаров, которое необходимо доставить по заказам (с учетом загружаемого на заказах и дополнительных пунктов загрузки) превышает емкость депо, то часть заказов будет отброшена.

Тот же принцип применяется по емкости депо по возврату товаров. Сумма товаров определенного вида, загруженных по заказам маршрутов, не может превышать емкости депо.

При первом использовании необходимо загрузить из учетной системы список товаров и установить показатели емкости депо.

Для этого необходимо нажать кнопку обновления таблицы на соответствующей вкладке, а затем скорректировать при необходимости емкость депо по конкретному виду товара - двойной клик на ячейке таблицы в колонке "Кол." переводит ячейку в режим прямого редактирования значения.

Если количество товаров конкретного вида по отгрузке или возврату превышает емкость депо и в системе указаны дополнительные пункты загрузки, емкость которых позволяет "догрузить" машину, то маршрут будет рассчитан с учетом посещения этих пунктов.

После нажатия "Enter" значение сохраняется в таблице.

По окончанию настройки реквизитов Главного депо нажмите кнопку "Сохранить" формы.

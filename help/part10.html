Основные понятия. Локальный поиск.

Оптимизация маршрутов транспортных средств - крайне трудоемкая задача, на решение которой налагается значительное количество условий и ограничений.

При большом количестве заказов поиск решения транспортной задачи путем простого перебора комбинаций превышает всякие практически допустимые временные рамки. Для решения подобного рода задач разработаны специальные алгоритмы, которые позволяют за относительно короткий промежуток времени найти приблизительно оптимальное решение.

Поэтому модуль оптимизации работает в двух вариантах поиска решения:

  • "быстрый поиск" (без установленного флага использования локального поиска), который позволяет предварительно настроить (подобрать) начальные параметры поиска решения;
  • локальный поиск, который позволяет проверить результаты за пределами "локального минимума" и улучшить первоначальное решение.

При "быстром поиске" программа завершает поиск решения на первом локальном минимуме. Локальный поиск позволяет достичь решения на втором (третьем и т.д.) локальных минимумах в пределах лимита времени для расчета.

Основные понятия. Груз.

Расчет каждого маршрута в режиме "Создание" или "Дополнение" учитывает количество перевозимого груза в каждом заказе и максимальную вместимость машины по видам груза.

Для целей расчета принимается, что количество отгруженного равно количеству заказанного.

Для того, чтобы транспорт эксплуатировался эффективно, программа должна найти решение, при котором будет найден кратчайший маршрут и при этом машина должна быть максимально загруженной. 

Кратчайшим маршрутом машины является случай, когда она вообще не выезжает за пределы депо. Максимальная же загрузка машины предполагает доставку груза максимальному количеству клиентов или, по крайней мере, клиентам с максимальными объемами заказа.

Вместимость машины.

В программе по каждой машине можно задать максимальную вместимость (по объему) для:

  • баллонов (ограничение по сумме количества баллонов всех видов);
  • воды в транспортных емкостях (предполагается, что перевозится только один вид);
  • торговых емкостей при установке или замене их у клиентов (предполагаем, что емкости любого вида занимают приблизительно одинаковое место в кузове машины).

Для других видов груза (ПЭТ, сифоны, подставки для торговых емкостей и т.п.) для упрощения расчета максимальное количество не устанавливается.

Для того, чтобы запретить распределение заказов на данную машину по какому-либо виду груза - просто установите значение 0(ноль) в соответствующем реквизите диалога.

Максимизация начальной загрузки.

Установка этих параметров указывает программе искать решение, при котором количество данного груза в начальной точке маршрута будет максимальным.

Для водовозов по-умолчанию установлена максимизация по воде в транспортных емкостях, для всех остальных - максимизация по баллонам.

суммарное количество товаров в отброшенных заказах (позволяет примерно оценить необходимое количество машин)

Основные понятия. Допустимые машины.

Доставка груза некоторым клиентам возможна только определенными машинами.

Например у клиента есть требования пропускного режима. Возможно размер или вес машины не позволяют осуществить доставку груза и т.п.

Для таких клиентов можно задать список "Допустимых машин". Если такой список не задан (пустой), то доставка возможна любой машиной.

Первоначально эти данные задаются в свойствах клиента в учетной системе:

Если у клиента существуют точки, до для каждой из них можно задать свой список допустимых машин:

Если для клиента задан список допустимых машин и точки установлен флаг "Наследовать от клиента", то при отгрузке на точку будет использован список допустимых машин клиента .

Если у точки сброшен флаг "Наследовать от клиента" и для точки задан свой список допустимых машин, то при отгрузке на точку будут использованы машины списка точки клиента.

Если у точки сброшен флаг "Наследовать от клиента" и список допустимых машин точки пустой,то отгрузка будет произведена любой машиной.

Список допустимых машин можно переопределить в диалоге редактирования заказа

или задать его списку заказов

Внимание!

Допустимые машины клиента имеют приоритет перед допустимыми группами клиентов машины!

Например, если для Исузу №1 установлены допустимые группы "Оптовики", "Предприятия", но при этом в заказе для клиента группы "Частные лица-Адреса" эта машина указана в качестве допустимой, то заказ может быть распределен на эту машину.

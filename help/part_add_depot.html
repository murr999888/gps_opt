Основные понятия. Пункты загрузки.

Промежуточные пункты загрузки - места загрузки / разгрузки, динамически при необходимости добавляемые в маршрут в случае, когда емкости главного депо недостаточно для отгрузки или возврата товаров маршрута. 

Необходимость посещения пункта погрузки определяется программой при расчете маршрута. Если для удовлетворения потребности в товаре заказов маршрута необходимо догрузиться товаром в промежуточном пункте, то в маршрут будет добавлен дополнительный заказ с посещением данного места и количеством погруженного товара равного необходимому. Отгрузка товара будет производиться с учетом емкости пункта загрузки - сумма погруженных в данном пункте товаров по всем маршрутам не может превышать емкости пункта загрузки по конкретному виду товаров.

В любом случае главное депо будет иметь приоритет по погрузке / возврату товаров. Если и главное депо и дополнительный пункт загрузки по отдельности имеют достаточную емкость конкретного товара для удовлетворения всех заказов, то загрузка машин этим товаром будет осуществляться только в главном депо.

Данные о дополнительных пунктах загрузки расположены на вкладке "Формирование МЛ"

Перед началом работы необходимо загрузить пункты загрузки из учетной системы - нажмите кнопку "Обновить" панели.

Для того, чтобы задействовать пункт загрузки в расчете, установите флаг "Использовать".

Для просмотра или редактирования реквизитов пункта - кликните два раза на нужной строке.

Форма редактирования реквизитов:

Реквизиты пункта доставки аналогичны реквизитам обычного заказа, за исключением времени работы (вместо окна доставки) и емкости загрузки / отгрузки (вместо заказа товаров)
